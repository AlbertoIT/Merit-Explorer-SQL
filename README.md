# Merit-Explorer-SQL

A tool made with Javascript and SQLite to run queries over the merit data on the forum Bitcointalk.

**Tables:**

```
MeritData(
  "Date" Integer,
  "Merit" Integer,
  "Msg" TEXT,
  "FromID" Integer,
  "ToID" Integer,
  "Board" TEXT,
  "SubBoard" TEXT
)
```

For more information and discussion: https://bitcointalk.org/index.php?topic=4551881.0
